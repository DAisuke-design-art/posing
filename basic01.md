AI画像生成（Stable Diffusion, Midjourney, Niji Journeyなど）において、グラビア特有のポーズを意図通りに出力させるには、AIが学習元（多くの場合はDanbooruなどのタグシステム）として使っている**「英語の共通タグ（プロンプト）」**を知ることが重要です。

日本語のニュアンスと、AIに確実に伝わる英語プロンプトの対応表を作成しました。これをそのままコピペして使用できます。

---

### 1. 全身・立ちポーズ系（Standing & Body Line）

身体のラインやひねりを表現するための必須プロンプトです。

| 日本語のイメージ | 英語プロンプト (推奨) | AIへの指示詳細・備考 |
| :--- | :--- | :--- |
| **見返り美人 / 振り向き** | **`looking back`, `from behind`, `back shot`** | 背中側から顔をこちらに向けている構図。 |
| **体をひねる / S字** | **`twisting torso`, `standing`** | 体をひねってラインを強調します。 |
| **壁に寄りかかる** | **`leaning against wall`** | 壁に背中や肩をつけている状態。 |
| **前かがみ** | **`leaning forward`, `bent over`** | 胸元を強調したり、挑発的な構図に使われます。 |
| **腰に手を当てる** | **`hands on hips`** | 自信のある立ち姿、くびれの強調。 |

### 2. 座りポーズ系（Sitting）

座り方は種類によって英語タグが明確に分かれています。

| 日本語のイメージ | 英語プロンプト (推奨) | AIへの指示詳細・備考 |
| :--- | :--- | :--- |
| **アヒル座り / ぺたん座り** | **`w-sitting`** | アルファベットのWの形に脚を開いて座るポーズ。 |
| **正座** | **`seiza`, `kneeling`** | `kneeling`は膝立ちも含むため、`seiza`が確実です。 |
| **体育座り** | **`sitting`, `hugging knees`** | 膝を抱えている状態。 |
| **足投げ出し / 開脚** | **`sitting`, `legs stretched out`, `spread legs`** | 足を前に伸ばして座る。`spread legs`は少し大胆になります。 |
| **片膝立て** | **`one knee up`** | 体育座りの片足版。奥行きが出ます。 |

### 3. 寝そべり・接地ポーズ系（Lying & Floor）

重力感やベッド上の表現に使います。

| 日本語のイメージ | 英語プロンプト (推奨) | AIへの指示詳細・備考 |
| :--- | :--- | :--- |
| **四つん這い / 猫ポーズ** | **`all fours`, `crawling`, `on hands and knees`** | 四つん這い。`ass up`を加えるとヒップが高くなります。 |
| **うつ伏せ** | **`lying on stomach`, `prone`** | お腹を下にして寝ている状態。 |
| **仰向け** | **`lying on back`, `supine`** | 背中を下にして寝ている状態。 |
| **横向きに寝る** | **`lying on side`** | 体の側面を見せる寝方。 |
| **バンザイ（寝た状態で）** | **`arms up`, `hands above head`** | 脇やウエストの細さを強調します。 |

### 4. 手の仕草・視線・表情（Gestures & Expressions）

グラビアらしさを高めるための「味付け」プロンプトです。

| 日本語のイメージ | 英語プロンプト (推奨) | AIへの指示詳細・備考 |
| :--- | :--- | :--- |
| **髪をかき上げる / 結ぶ** | **`adjusting hair`, `arms behind head`, `tying hair`** | 脇や二の腕を見せる効果があります。 |
| **谷間強調 / 胸寄せ** | **`squeezing breasts`, `cleavage`** | 腕で寄せる場合は`arms under boobs`も有効。 |
| **カメラ目線** | **`looking at viewer`** | 基本中の基本。これがないと目線がズレやすいです。 |
| **上目遣い** | **`looking up`** | カメラが上にある（ハイアングル）時によく併用します。 |
| **恥じらい / 赤面** | **`embarrassed`, `blush`** | 頬を赤らめる表現。 |

---

### 5. カメラアングル（重要）

ポーズと同じくらい重要なのが「カメラの位置」です。これを指定するとクオリティが劇的に上がります。

* **ローアングル（下から）:** `low angle`, `from below`
    * 脚を長く見せたい、威圧感や迫力を出したい時に有効。
* **ハイアングル（上から）:** `high angle`, `from above`
    * 上目遣いや、胸元を強調したい時、か弱さを出したい時に有効。
* **太ももから上:** `cowboy shot`
    * 膝上から頭までを入れる、グラビアで最もバランスの良い構図。
* **顔のアップ:** `portrait`, `close-up`
* **被写界深度（背景ボケ）:** `depth of field`, `blurry background`
    * 人物にピントを合わせ、背景をボカしてプロっぽい写真にします。

### 実践的なプロンプト構成例

これらを組み合わせると、以下のような呪文（プロンプト）になります。

> **例：ビーチでアヒル座りをして、上目遣いでこちらを見ている水着**
> `1girl, solo, bikini, w-sitting, on beach, high angle, looking at viewer, looking up, hands on ground, depth of field, photorealistic`

### 💡 アドバイス：強調と重み付け

AI画像生成では、特定の要素を強調したい場合、カッコ `( )` や数字を使います。
* **`(w-sitting:1.2)`** -> アヒル座りの要素を1.2倍強くする。
* **`{{looking back}}`** -> Niji Journeyなどで強調する場合。

作りたい具体的なイメージ（例：「夕暮れの教室で」「濡れたシャツで」など）があれば、それに合わせたプロンプトの組み合わせも提案できます。作成したいシチュエーションはありますか？
